### class(类)
  1. 面向对象编程
    * 类
    * 对象
    * 方法
    * 属性
    * 继承
    * 实例化
    * 构造方法
      1. 使用new实例化对象的时候，自动调用构造方法
      2. 一个类必须有构造方法
  2. ES6 引入 class的概念
  3. 继承
     * 一个类继承于另一个类，并且拥有另一个类的属性和方法
     * ES6 使用 extends 关键字 继承
     * super关键字在构造方法里调用，
     表示调用父类的 构造方法
     * 先使用super，才可以使用this
  4. 类的静态方法
     * 类中定义的方法，都会被实例继承
     * 如果在方法中加上 static 关键词，则表示不会被继承
     * 静态方法 直接通过类调用
     * 父类的静态方法，可以被子类继承
  5. 静态属性
     * 定义在class类上的属性，而不是实例化对象上的属性
